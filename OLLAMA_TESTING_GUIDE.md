# 🤖 בדיקת כל מודלי Ollama לעברית

## מטרה
לבדוק את **כל** מודלי Ollama הזמינים ולמצוא מי הכי טוב לעברית.

## איך להריץ

### אופציה 1: דרך GitHub Actions (מומלץ!)

1. עבור ל-GitHub → Actions
2. בחר "Test All Ollama Models for Hebrew"
3. לחץ "Run workflow"
4. בחר מודלים:
   - השאר ריק לברירת מחדל
   - או הכנס רשימה: `aya-expanse:8b,llama3.2:3b,gemma2:9b`

### אופציה 2: מקומית

```bash
python test_all_models.py "aya-expanse:8b,llama3.2:3b,gemma2:9b"
```

## מה הבדיקה בודקת?

4 שאלות בסיסיות:

1. **מצות - אוכלים** (צריך: לחם)
2. **מצות - תורה** (צריך: מצוות)
3. **ב"ה בתחילה** (צריך: ברוך ה')
4. **ב"ה בסוף** (צריך: ברוך הוא)

## שימוש בתוצאות

לאחר הבדיקה:
1. בחר את המודל עם הדיוק הגבוה ביותר
2. שקול גם מהירות (זמן ממוצע)
3. השווה ל-DictaBERT (95%+)
4. זכור: DictaBERT ייעודי לעברית ולכן הכי טוב!

## הערות

- הבדיקה לוקחת זמן (כל מודל = 2-5 דקות)
- GitHub Actions מאפשר עד 6 שעות
- מומלץ לבדוק 5-10 מודלים בפעם
- התוצאות יישמרו כ-artifact
